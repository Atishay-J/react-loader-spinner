# Getting Started

### Installation

```
$ npm install react-loader-spinner --save

#or

$ yarn add react-loader-spinner
```

## Usage

```jsx

import { Audio } from  'react-loader-spinner'

<Audio
    height = "80"
    width = "80"
    radius = "9"
    color = 'green'
    ariaLabel = 'three-dots-loading'     
    wrapperStyle
    wrapperClass
  />
```

## Features 🎸 🚀 👽:

- Typescript Support
- Customizable SVG Loaders
- Light Weight
